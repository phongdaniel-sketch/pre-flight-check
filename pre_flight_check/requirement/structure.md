## Proposed System Architecture

| Component | Technology Information | Notes |
| :--- | :--- | :--- |
| **Frontend** | **Vue.js** | Modern SPA framework for component-based UI. |
| **Backend** | **Node.js** (Express) | Unified JS/TS stack for API & Integrations. |
